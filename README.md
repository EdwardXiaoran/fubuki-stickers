## Requirements

+ Node.js
+ thumbsup
+ Exiftool
+ GraphicsMagick
+ Gifsicle

## How to contribute

直接在`/fubuki/`下面添加新的表情包即可，支持gif，然后Pull Request即可，为避免重复**请将文件重命名为其MD5值**（小写）.
